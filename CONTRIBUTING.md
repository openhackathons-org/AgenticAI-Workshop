# Contributing

Thanks for helping improve the Agentic AI workshop technical guide.

## Suggested Workflow

1. Open an issue for additions, corrections, or unclear sections.
2. Use a pull request for changes to `india-agentic-ai-open-hackathon-technical-guide.html`.
3. Keep each pull request focused on one topic where possible.
4. Preview the HTML file in a browser before requesting review.

## Content Guidelines

- Keep guidance practical for hackathon developers.
- Prefer official NVIDIA and OpenHackathons links.
- Call out preview or alpha software clearly.
- Make track-specific guidance easy to scan.
