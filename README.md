# AgenticAI Workshop

Technical guide for developers participating in the India Agentic AI Open Hackathon workshop.

## Review The Guide

Open the guide here:

- [India Agentic AI Open Hackathon Technical Guide](india-agentic-ai-open-hackathon-technical-guide.html)

For a rendered local preview, download or clone this repository and open `india-agentic-ai-open-hackathon-technical-guide.html` in a browser.

## What This Repository Is For

Use this repository to review, coordinate, and improve the developer-facing technical guide before and during the workshop.

The guide covers:

- NVIDIA NIM and Nemotron as the common foundation.
- Track A: Agentic workflows.
- Track B: Model finetuning and customisation.
- Track C: Synthetic data generation.
- Readiness checklists and official links.
